📌 Next Word Prediction using NLP
📖 Project Overview

This project focuses on building a Next Word Prediction model using Natural Language Processing (NLP) techniques.
The goal of the model is to predict the most probable next word in a sentence based on the previously entered words.

Next word prediction is widely used in applications such as:

Smart keyboards

Search engines

Chatbots

Text auto-completion systems

🎯 Objective

Build a model that can predict the next word in a sequence

Learn patterns and relationships between words in a text dataset

Implement deep learning techniques for sequence modeling

🗂 Dataset

The dataset used for this project contains movie descriptions and text data.

Example sentence:

Input:

The hero fights the

Output:

villain

🧠 Model Architecture

The model is built using a Deep Learning approach:

Text preprocessing

Tokenization

Sequence generation

Padding sequences

Training an LSTM Neural Network

Model Layers:

Embedding Layer

LSTM Layer

Dense Layer with Softmax activation

1️⃣ Data Cleaning

5️⃣ Training the Model
6️⃣ Predicting the Next Word
